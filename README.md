# BootCamp Renda Extra com IA

Site estático publicado no GitHub Pages.

## Como publicar

1. Faça commit e push do repositório.
2. No GitHub, vá em Settings → Pages.
3. Selecione "Deploy from a branch".
4. Escolha a branch `main` e a pasta `/root`.
5. Salve.

> O arquivo [.nojekyll](.nojekyll) foi adicionado para evitar problemas com arquivos estáticos.
